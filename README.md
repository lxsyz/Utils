# Utils
整理收藏常用的工具类
AppUtil 获取应用程序名称版本号（持续更新）
FileUtil 检测SDcard是否挂载
	 生成图片名称
	 根据文件路径压缩图片返回bitmap
	 压缩图片生成新的文件
	 bitmap转换成文件
	 调用手机已安装程序打开对应类型的文件
KeyBoardUtil 键盘的开启关闭
LogerHelper  日志工具类
SPUtil   SharedPreference工具类
ToastUtil Toast工具类
ScreenUtil  获取屏幕宽高
	    获取屏幕截图等
OperatorUtil    简单操作类
		* 1. 判断是否是email格式
 		* 2. 判断是否是手机号码
 		* 3. dp2px   px2dp
 		* 4. 获取汉字的首字母
 		* 5. 计算两段时间相差多少秒 例如 下午3点  下午5点  "下午HH点"
 		* 6. 判断是否联网
 		* 7. 判断是否是wifi  3g等等
 		* 8. SHA1加密算法
